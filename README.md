# Telecom Customer Churn Analysis

## Project Overview

This project analyzes telecom customer data to identify patterns and factors that contribute to customer churn. The goal is to help businesses understand why customers leave and provide data-driven insights to improve retention strategies.

The analysis includes data cleaning, KPI creation, and interactive dashboard visualizations.

---

## Objectives

* Identify key factors influencing customer churn
* Build meaningful KPIs for churn monitoring
* Create an interactive dashboard for business insights

---

## Dataset Description

The analysis is based on the Customer_Data table, which includes:

* **Demographics**: Gender, Age, State, Married status.
* **Services**: Phone Service, Internet Type (DSL, Fiber Optic, Cable), Online Security, Streaming.
* **Account Info**: Tenure, Contract Type, Payment Method, Monthly Charges.
* **Churn Status**: Churn Category (Competitor, Price, Attitude, etc.) and Churn Reason.

## Key Performance Indicators

Based on the analyzed dataset of 6,418 total customers:

* **Total Churn**: 1,732 customers
* **Churn Rate**: ~27%
* **New Joiners**: 411 customers

---

## Key Insights

The data reveals several high-risk segments that require immediate attention:

* **Competitor Pressure**: 761 customers (the largest group) left for a competitor.

* **Contract Vulnerability**: Customers on Month-to-Month contracts have a significantly higher churn rate (46.5%) compared to those on two-year contracts (2.7%).

* **Service Issues**: Customers using Fiber Optic internet have the highest churn rate at 41.1%, suggesting possible service or pricing dissatisfaction in that segment.

* **Demographic Risk**: Senior citizens (Age 61+) show a churn rate of 47.5%, which is nearly double the average.
